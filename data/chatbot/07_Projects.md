## GenAI SQL Assistant
**Self-Service Analytics Platform** | Feb 2024 – Apr 2024 | Self-led

### Solution Architecture
Built a comprehensive self-service GenAI analytics assistant specifically designed to empower non-technical users to query structured data using intuitive natural language interfaces. This democratizes data access and accelerates decision-making across diverse stakeholder groups.

### Technical Innovation
Seamlessly integrated OpenAI's advanced language models with Streamlit framework to create real-time natural language-to-SQL translation capabilities. The system interprets user questions in plain English and automatically generates optimized SQL queries against structured datasets.

### Business Impact
Eliminated technical barriers for business users seeking data insights, enabling self-service analytics capabilities that traditionally required SQL expertise. Accelerated decision-making processes by providing instant access to data insights through conversational interfaces.

### Visualization & Insights
Implemented real-time data visualization features that automatically generate charts, graphs, and summary statistics based on query results. Users receive both raw data and interpreted insights to support informed decision-making.

**Technologies Used:** Python, OpenAI API, Streamlit, SQLite, Natural Language Processing

## InsightMate: AI-Powered Data Exploration
**Intelligent Data Analysis Platform** | Feb 2024 – Apr 2024 | Self-led

### Platform Capabilities
Developed InsightMate, an advanced GenAI assistant that enables users to analyze uploaded datasets through plain English queries without requiring SQL knowledge or coding expertise. This solution democratizes data analysis for business users across all technical skill levels.

### Advanced Query Processing
Implemented sophisticated query conversion system using GPT-4 to translate natural language questions into optimized DuckDB SQL statements. The system handles complex analytical queries including aggregations, filtering, joins, and statistical calculations.

### Interactive Visualization Engine
Integrated comprehensive charting capabilities using Plotly to automatically generate appropriate visualizations based on query results and data types. Users receive both tabular results and visual representations of their data insights.

### Automated Insight Generation
Built intelligent trend analysis features that provide automatic explanations of data patterns, anomalies, and statistical significance. The system delivers contextual insights alongside raw analysis results.

**Technologies Used:** GPT-4, Streamlit, DuckDB, Plotly, Pandas, Data Visualization

**Links:**
- [Live Demo](https://ags-purdue-insightmate.streamlit.app/)

## AI Stock Analyzer
**Financial Intelligence Platform** | Feb 2024 – Jun 2024 | Self-led

### Comprehensive Financial Analysis
Created an sophisticated AI-powered financial insights tool offering sector-based stock discovery, GPT-generated investment recommendations, and comprehensive trend visualizations. The platform integrates multiple data sources to provide holistic market analysis.

### Real-Time Market Integration
Implemented real-time market data integration using yFinance API to provide current stock prices, historical performance data, and technical indicators. The system continuously updates analysis based on latest market conditions.

### Advanced Sentiment Analysis
Integrated comprehensive sentiment analysis capabilities using Feedparser to analyze financial news, social media, and market commentary. This provides additional context for investment decisions beyond traditional technical analysis.

### Interactive Dashboard Experience
Built sophisticated interactive dashboards using Plotly that enable users to explore stock performance, compare sectors, analyze trends, and visualize portfolio scenarios. The interface supports both novice and experienced investors.

### AI-Powered Recommendations
Leverages GPT-4 to generate personalized investment recommendations based on user preferences, risk tolerance, market conditions, and sentiment analysis. Provides rationale and supporting data for each recommendation.

**Technologies Used:** GPT-4, Streamlit, yFinance, Plotly, Feedparser, Sentiment Analysis, Financial APIs

## Personalized Learning AI Recommender
**Intelligent Educational Content Curation** | Jun 2024 | Self-led

### Autonomous Learning Discovery
Created a sophisticated Personalized Learning AI Recommender that autonomously discovers and curates relevant online learning resources tailored to individual user goals, preferred learning styles, and current skill levels. This agentic AI application transforms the learning discovery process.

### Advanced AI Agent Architecture
Implemented comprehensive agentic AI system that autonomously perceives user learning objectives and web content through intelligent parsing. The system reasons via LLM-powered search algorithms, content reranking, and comprehensive content assessment to optimize resource selection.

### Intelligent Information Pipeline
Designed and implemented optimized information retrieval pipeline that plans and executes sophisticated web scraping, content analysis, and relevance scoring. The system acts autonomously to deliver curated, transparent, and highly personalized learning resources.

### Multi-Source Content Integration
Integrated diverse content sources through web scraping (BeautifulSoup, Requests), API integration with educational platforms, and Google Custom Search to provide comprehensive learning resource coverage across multiple domains and formats.

### Personalization Engine
Built advanced personalization algorithms that adapt to user preferences, learning progress, and feedback to continuously improve recommendation quality and relevance over time.

**Technologies Used:** Python, Gradio, BeautifulSoup, Requests, Anthropic API, Google Custom Search API, SQLite, Web Scraping
